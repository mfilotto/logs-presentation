Splunk - Pros


  1. Patterns de reconnaissance de logs très performants ex: 100% sur Apache
  2. Configuration entièrement dans la console web
  3. Nombreux modèles de données disponibles (CIM: Common Information Model)
  4. Très bonne assistance à la construction de requêtes : auto-suggestion inteligente

Splunk - Cons


  1. Console web pas toujours très intuitive : agregation (sum) difficiles à trouver, fonction avancer comme GeoIP non trouvée
  2. Trop monolithique
  3. Prix excessif
  4. Apps (plugin) : peu robustes
  5. Useragent pas optimisé et inopérant

The Elastic Product Portfolio


ELK stack - Pros


  1. Architecture très scalable, flexible et modulaire
  2. Système très ouvert
  3. Chaque brique adresse une fonction très spécifique
  4. Compétences exploitables dans d'autres cas d'utilisation
  5. Très grande communauté et adoption
  6. ES : Ecosystème dynamique, nombreux outils autour de ES
  7. ES : très performant, robuste, scalable, très rapide
  8. Logstash : filtres intégrés bien pratiques : geoip, useragent
  9. Kibana : facile à prendre en main, intuitif
  10. Kibana : Plage temporelle facile d'utilisation

ELK stack - Cons


  1. Difficile de trouver des configurations toutes faites
  2. Configuration à réaliser soit même
  3. Configuration manuelle pour la collecte
  4. Patterns groks ne couvrent pas tous les cas ex pour apache ou 5 logs sur 5888 passent au travers à cause d'un problème de parsing
  5. Certaines fonctions clef sous licence : gestion des accès, gestion des watchers

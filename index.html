Splunk - Pros


  1. Patterns de reconnaissance de logs très performants ex: 100% sur Apache
  2. Configuration entièrement dans la console web
  3. Nombreux modèles de données disponible (CIM: Common Information Model)

Splunk - Cons


  1. Console web pas toujours très intuitive : agregation (sum) difficiles à trouver, fonction avancer comme GeoIP non trouvée
  2. Trop monolithique
  3. Prix excessif

The Elastic Product Portfolio


ELK stack - Pros


  1. ES très performant, robuste, scalable, très rapide
  2. Très modulaire, chaque brique adresse une fonction très spécifique, possibilité de réutilisations dans d'autres cas d'utilisation
  3. Très grande communauté et adoption
  4. Ecosystème dynamique, nombreux outils autour de ES
  5. Système très ouvert
  6. Kibana facile a prendre en main, intuitif
  7. Des filtres logstash bien pratique : geoip, useragent, faciles à intégrer
  8. Architecture très scalable et très flexible
  9. Plage temporelle facile d'accès

ELK stack - Cons


  1. Patterns groks ne couvrent pas tous les cas ex pour apache ou 5 logs sur 5888 passent au travers à cause d'un problème de parsing
  2. Difficile de trouver des configurations toutes faites
  3. Configuration à réaliser soit même
  4. Configuration manuelle pour la collecte
  5. Des fonctions sous licence : gestion des accès, gestion des watchers
